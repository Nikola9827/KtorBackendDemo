# KtorBackendDemo

A Kotlin Ktor backend demonstrating a simple REST API with one endpoint. Returns dynamic messages to clients (like AndroidKtorClient). Designed for learning and full-stack demonstration.

**Tech stack:** Kotlin, Ktor, Coroutines, MVVM (frontend client), Dependency Injection (Koin)

**Features:**
- Exposes a `GET /helloworld` endpoint returning current time.
- Works seamlessly with AndroidKtorClient.
- Easy to extend for additional endpoints.
